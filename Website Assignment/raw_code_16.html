Code Explanation:
  • The program uses a do-while loop to print numbers from 1 to 10.
  • It starts with count set to 1.
  • The do block runs first and prints the current value of count.
  • It then increments count by 1.
  • After each execution, the condition count <= 10 is checked. If true, the loop continues.

(Note: A do-while loop always runs the code at least once before checking the condition.)
